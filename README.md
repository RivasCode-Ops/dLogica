# dLogica

dLogica é um app para transformar uma necessidade confusa em uma definição clara de problema, objetivo, critérios, decisão e solução recomendada.

## Objetivo

Organizar, filtrar, comparar e decidir antes de construir, reduzindo retrabalho e decisões fracas.

## Documento base

O PRD inicial completo está em `dlogica/PRD-INICIAL.md`.

## Arquivos operacionais

- `CONTEXT.md`: estado atual e direção em vigor.
- `HANDOFF.md`: continuidade entre sessões/agentes.
- `PRD-INICIAL.md`: escopo funcional e regras do produto.

## Próximo passo recomendado

Executar uma primeira rodada de validação com 3 demandas reais e registrar:
- clareza da saída final;
- tempo até decisão;
- qualidade da recomendação de tipo de solução.
